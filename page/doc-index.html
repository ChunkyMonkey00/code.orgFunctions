onEvent(elem, event, func)

On event takes 3 arguments. This function is a part of the App lab in code.org. The elem/element argument takes string as an input and represents the element being modified. The event arguments takes string as an input and represents the event that the given element is watching for. The func argument represents the function to be ran when the event is detected.

This function's purpose is to watch for events on given elements and run a function when the event is detected.

setProperty(elem, prop, val)

Set property takes 3 arguments. This function is a part of the App lab in code.org. The elem/element argument takes string as an input and represents the element being modified. The prop/property argument takes string as an input and represents what property is being modified. PLEASE NOTE: if the property is not a base HTML attribute, (generally the case for CSS attributes/styling such as border) you must add 'style.' before the property to be able to modify/access it. The val/value property argument takes string/numbers as an input and represents the value that a property is being changed to.

The purpose of set property is to set a property of a given element.

getProperty(element, property)

Get property takes 2 arguments. This function is a part of the App lab in code.org. The elem/element argument takes string as an input and represents the element being modified. The prop/property argument takes string as an input and represents what property is being modified. PLEASE NOTE: if the property is not a base HTML attribute, (generally the case for CSS attributes/styling such as border) you must add 'style.' before the property to be able to modify/access it.

The purpose of get property is to get a given property of a given element.

getText(element)

Get text takes 1 arguments. This function is a part of the App lab in code.org. The elem/element argument takes string as an input and represents the element being modified.

The purpose of the get text function is to get the text content of a given element.
